# EDUVISION

AI-Driven Classroom Analytics and Attendance Monitoring System

## Overview

EDUVISION is an AI-powered classroom monitoring system that automates attendance verification and classroom analytics using facial recognition and computer vision.

## Features

- Facial Recognition Attendance
- Sleeping Posture Detection
- Yawning Detection
- Phone Usage Detection
- Noise Monitoring
- Unknown Person Detection
- Teacher Dashboard
- Admin Dashboard
- Attendance Reports

## Technologies Used

- Python
- Flask
- OpenCV
- SQLite
- ESP32
- HTML
- CSS
- JavaScript
