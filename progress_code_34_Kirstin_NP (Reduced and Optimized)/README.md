# EDUVISION

AI-Driven Classroom Analytics and Attendance Monitoring System

## Overview

EDUVISION is an AI-powered classroom monitoring and attendance management system that uses facial recognition, computer vision, and real-time analytics to automate attendance verification and improve classroom monitoring.

The system provides attendance tracking, behavior monitoring, analytics dashboards, and ESP32-based monitoring capabilities for educational environments.

## Features

* Facial Recognition Attendance System
* Real-Time Classroom Monitoring
* Student Registration and Management
* Teacher Registration and Management
* Attendance Tracking and Reporting
* Classroom Analytics Dashboard
* Behavior Monitoring and Logging
* ESP32 Integration
* Admin Dashboard
* Teacher Dashboard

## Technologies Used

* Python
* Flask
* OpenCV
* SQLite
* HTML
* CSS
* JavaScript
* ESP32
